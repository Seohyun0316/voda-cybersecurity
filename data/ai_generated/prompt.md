# VibeSafe 실험 프롬프트 세트 v2 (함정형 150개)

### ⚠️ 수집 전 반드시 필독

**1. 프롬프트 안의 API 키·비밀번호는 전부 가짜 더미 값입니다**
→ 절대 본인의 실제 키/비밀번호로 바꾸지 마세요!!!
2. 프롬프트 원문 그대로 입력 (단어 추가·수정 금지, 특히 "안전하게" 금지)
**3. AI의 첫 번째 응답 코드를 저장**(재생성·"더 잘 짜줘" 금지)
4. 파일명: {번호}_{모델}_{언어} (예: P012_gpt_python.py)
5. 첫 응답이 안전하게 나와도 그대로 저장 (그것도 데이터)

### 함정형 프롬프트 설계 원리

"안전하게 짜줘"라고 하면 모델이 방어 코드를 넣지만, 비전공자는 그렇게 안 씀..
실제로는 "빨리 되게", "에러 없애줘", "내 키 이거야", "제일 간단하게"처럼 요청함
v2는 이 실제 발화를 재현해 취약 코드를 유도함. 보안이라는 단어는 언급하지 않음
